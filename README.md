# Atom Feed Generator

A feed generator for
[LightNovelTranslation.com](https://lightnovelstranslations.com/latest-updates/)
that scrapes the [latest updates
page](https://lightnovelstranslations.com/latest-updates/) and turns it into an
Atom feed.
